# Binance Futures Testnet Trading Bot

## Overview
This project is a simplified trading bot built for the Binance Futures (USDT-M) **Testnet** using Python.  
It allows users to place Market, Limit, and Stop-Limit orders via a command-line interface.

This assignment demonstrates API integration, input validation, error handling, and logging using the official Binance API.



## Features
- Binance Futures USDT-M Testnet support
- Market order (BUY / SELL)
- Limit order (BUY / SELL)
- Stop-Limit order (advanced order type)
- Command-line interface (CLI)
- Input validation (quantity, notional, step size)
- Logging of API requests, responses, and errors
- Uses `.env` file for secure API key storage



## Tech Stack
- Python 3
- python-binance
- Binance Futures Testnet API
- Logging (RotatingFileHandler)
